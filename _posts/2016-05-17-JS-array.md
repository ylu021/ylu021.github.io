# 数组
### 创建
1. `[]`
2. `[1,,2]`//1,undefined,2
3. `new Array()`
  - 传入一个参数是length
  - 传入N个参数就是元素
  - new可以省略  

> delete 删除数组指定的元素，实际上将它用undefined代替，length不变

### 添加/删除
```
.push() - add back
.unshift() - add front
.pop() - delete back
.shift() - delete front
```

### 种类
二维  
稀疏 - 不连续`[,,]`  
  判断用in  
  arr1 = [undefined]
  0 in arr1 //true 因为下标0的key值是undefined  
  arr2 = new Array(1)
0 in arr2 //false 因为下标0的key值不存在
